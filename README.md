Aplikasi Manajemen Produk Berbasis PHP OOP

Aplikasi ini merupakan aplikasi web sederhana yang dikembangkan menggunakan PHP berbasis Object Oriented Programming (OOP) dan konsep pemrograman modular. Sistem ini menerapkan arsitektur MVC sederhana, routing menggunakan .htaccess, serta sistem autentikasi dengan role admin dan user.

Aplikasi dibuat sebagai bagian dari project praktikum OOP, dengan tujuan memahami penerapan konsep OOP, routing, dan pengelolaan data pada aplikasi web.
Fitur Aplikasi
1. Autentikasi dan Otorisasi

Sistem login menggunakan username dan password

Password disimpan menggunakan fungsi password_hash()

Autentikasi berbasis session

Pembagian hak akses berdasarkan role:

Admin

User

2. Manajemen Produk (Admin)

Menambah data produk

Mengubah data produk

Menghapus data produk

Menampilkan daftar produk

Upload foto produk

Pencarian data produk

Pagination pada daftar produk

3. Tampilan Antarmuka

Desain responsive dengan pendekatan mobile-first

Menggunakan framework CSS Bootstrap

Tampilan sederhana dan mudah digunakan

Teknologi yang Digunakan

PHP Native (Object Oriented Programming)

MySQL dengan PDO

HTML, CSS, dan JavaScript

Bootstrap

Apache Web Server (.htaccess untuk routing)
